# Menu

A menu is used to display a list of options to the user. It appears when the user interacts with a button, link, or other control that triggers an action.

## Import

- `Menu`: The component that provides the menu functionality.
- `MenuTrigger`: The trigger that opens the menu. This is usually a button or link.
- `MenuButton`: A button that composes `MenuTrigger` and `Button` and is used to open the menu.
- `MenuIndicator`: An indicator that appears next to the trigger. This is usually a chevron or arrow.
- `MenuList`: A wrapper that contains the menu items. Must be a direct child of the `Menu` component.
- `MenuGroup`: A wrapper that groups related menu items.
- `MenuItem`: The individual menu items. Must be a direct child of the `MenuList` component.
- `MenuDivider`: A divider that separates menu items.

```js
import {
  Menu,
  MenuTrigger,
  MenuButton,
  MenuIndicator,
  MenuList,
  MenuGroup,
  MenuItem,
  MenuDivider,
} from '@tonic-ui/react';
```

## Usage

### Basic menu structure

Below is a simple example that shows the basic menu structure.

```jsx
<>
  <MenuItem value="general">
    <Icon icon="settings" mr="2x" />
    <Text>General</Text>
  </MenuItem>
  <MenuItem value="accounts">
    <Icon icon="user-team" mr="2x" />
    <Text>Accounts</Text>
  </MenuItem>
  <MenuItem value="privacy">
    <Icon icon="lock" mr="2x" />
    <Text>Privacy</Text>
  </MenuItem>
  <MenuDivider />
  <MenuItem disabled value="restore-defaults">
    <Icon icon="undo" mr="2x" />
    <Text>Restore Defaults</Text>
  </MenuItem>
</>
```

### Controlled menu

The menu opens over the anchor element by default, and closes when the user clicks outside of the menu.

```jsx
function Example() {
  const [anchorEl, setAnchorEl] = React.useState(null);
  const [isOpen, setIsOpen] = React.useState(false);
  const [selectedValue, setSelectedValue] = React.useState(null);
  const handleOpenMenu = (event) => {
    setAnchorEl(event.currentTarget);
    setIsOpen(!isOpen);
  };
  const handleCloseMenu = () => {
    setAnchorEl(null);
    setIsOpen(false);
  };
  const handleClickMenuItem = (event) => {
    const value = event.target.getAttribute('value');
    setSelectedValue(value);
  };
  const handleMenuItemClick = (event) => {
    const value = event.target.getAttribute('value');
    console.log(`Menu item #${value} clicked`);
  };

  return (
    <>
      <Button
        aria-controls="basic-menu"
        aria-haspopup="true"
        aria-expanded={isOpen ? 'true' : undefined}
        onClick={handleOpenMenu}
      >
        Options
      </Button>
      <Menu
        id="basic-menu"
        anchorEl={anchorEl}
        isOpen={isOpen}
        onClose={handleCloseMenu}
        mr="2x"
      >
        <MenuList
          width="max-content"
          onClick={handleClickMenuItem}
        >
          <MenuItem value={1}>
            Item #1
          </MenuItem>
          <MenuItem value={2}>
            Item #2
          </MenuItem>
          <MenuItem value={3}>
            Item #3
          </MenuItem>
        </MenuList>
      </Menu>
      <Space width="2x" />
      <Text>Selected: {selectedValue}</Text>
    </>
  );
}
```

`MenuTrigger` is the trigger that opens the menu when clicked. It is usually a button or link. The `MenuIndicator` can be added to the trigger to indicate the menu is open or closed.

### MenuButton

`MenuButton` composes `MenuTrigger` and `Button` to create a button that opens the menu. It's an useful alternative to `MenuTrigger` when the trigger is a button.

```jsx disabled
const MenuButton = (props) => (
  <MenuTrigger>
    {({ getMenuTriggerProps }) => (
      <Button
        {...getMenuTriggerProps()}
        {...props}
      >
        {children}
      </Button>
    )}
  </MenuTrigger>
);
```

### MenuIndicator

`MenuIndicator` is an indicator that indicates the menu is open or closed.

```jsx
<Menu>
  <MenuButton variant="ghost" size="sm" px="1x">
    <MenuIndicator />
  </MenuButton>
  <MenuList width="max-content">
    <MenuItem>
      Item #1
    </MenuItem>
    <MenuItem>
      Item #2
    </MenuItem>
    <MenuItem>
      Item #3
    </MenuItem>
  </MenuList>
</Menu>
```

You can also customize the indicator by passing a component as children or a function as children that returns a component.

```jsx
<Menu placement="bottom-start">
  {({ isOpen, direction }) => {
    return (
      <>
        <MenuButton variant="secondary">
          <MenuIndicator>
            {({ ref, style: styleProps }) => {
              // direction is either 'up' or 'down'
              const icon = `arrow-${direction}`;
              styleProps.transform = isOpen ? 'scaleY(-1)' : 'scaleY(1)';

              return (
                <Icon
                  ref={ref}
                  icon={icon}
                  size="3x"
                  {...styleProps}
                />
              );
            }}
          </MenuIndicator>
        </MenuButton>
        <MenuList width="max-content">
          <MenuItem>
            Item #1
          </MenuItem>
          <MenuItem>
            Item #2
          </MenuItem>
          <MenuItem>
            Item #3
          </MenuItem>
        </MenuList>
      </>
    );
  }}
</Menu>
```

### Menu sizing

To change the width of the menu, pass `width` prop to the `MenuList` component.

#### `width="100%"`

```jsx
<Menu>
  <MenuButton variant="secondary">
    <Text>This is a very long menu button</Text>
    <Space width="1x" />
    <MenuIndicator />
  </MenuButton>
  <MenuList width="100%">
    <MenuItem>
      Item #1
    </MenuItem>
    <MenuItem>
      Item #2
    </MenuItem>
    <MenuItem>
      Item #3
    </MenuItem>
  </MenuList>
</Menu>
```

#### `width="max-content"`

By default, the menu items will wrap to the next line if the width of the trigger is smaller than the width of the menu. You can pass `width="max-content"` to `MenuList` to prevent wrapping.

```jsx
<Menu>
  <MenuButton variant="secondary">
    <Text>Options</Text>
    <Space width="1x" />
    <MenuIndicator />
  </MenuButton>
  <MenuList width="max-content">
    <MenuItem>
      This is a very long menu item #1
    </MenuItem>
    <MenuItem>
      This is a very long menu item #2
    </MenuItem>
    <MenuItem>
      This is a very long menu item #3
    </MenuItem>
  </MenuList>
</Menu>
```

### Placement

To position the menu relative to the trigger, pass a `placement` prop to `Menu`.

The value can be one of the following:
* `top`
* `top-start`
* `top-end`
* `bottom`
* `bottom-start`
* `bottom-end`

```jsx noInline
const useSelection = (defaultValue) => {
  const [value, setValue] = React.useState(defaultValue);
  const changeBy = (value) => () => setValue(value);
  return [value, changeBy];
};

const gridAreas = [
  ['top-start', 'top', 'top-end'],
  ['.', '.', '.'],
  ['.', 'center', '.'],
  ['.', '.', '.'],
  ['bottom-start', 'bottom', 'bottom-end'],
];

function Example() {
  const [placement, changePlacementBy] = useSelection('bottom-start');
  const gridTemplateAreas = gridAreas.map((row) => {
    const rowString = '"' + row.join(' ') + '"';
    return rowString;
  }).join(' ');

  return (
    <Flex>
      <Grid
        templateAreas={gridTemplateAreas}
        autoColumns="minmax(0, 1fr)"
        autoFlow="row"
        gap="2x"
      >
        {gridAreas.flat().map((value, key) => {
          if (value === '.') {
            return (<Box key={key} />);
          }

          if (value === 'center') {
            return (
              <Box key={key}>
                <Menu placement={placement}>
                  <MenuButton variant="secondary">
                    <Text>Options</Text>
                    <Space width="1x" />
                    <MenuIndicator />
                  </MenuButton>
                  <MenuList width="max-content">
                    <MenuItem>
                      Item #1
                    </MenuItem>
                    <MenuItem>
                      Item #2
                    </MenuItem>
                    <MenuItem>
                      Item #3
                    </MenuItem>
                  </MenuList>
                </Menu>
              </Box>
            );
          }

          const changePlacement = changePlacementBy(value);
          const onClick = () => {
            changePlacement();
          };

          return (
            <Box key={key}>
              <SelectableButton
                selected={value === placement}
                onClick={onClick}
                width="100%"
              >
                {value}
              </SelectableButton>
            </Box>
          );
        })}
      </Grid>
    </Flex>
  );
}

render(<Example />);
```

### Offset

The basic offset accepts an array with two numbers in the form `[skidding, distance]`. It can be used to adjust the position of the menu relative to the trigger.

**Skidding**

The first number, `skidding`, displaces `MenuList` along the reference trigger. This value is specified in pixels.

**Distance**

The second number, `distance`, controls the distance between the `MenuList` and the reference trigger. This value is specified in pixels.

```jsx
<Menu>
  <MenuButton variant="secondary">
    <Text>Options</Text>
    <Space width="1x" />
    <MenuIndicator />
  </MenuButton>
  <MenuList
    offset={[
      16, // skidding
      8, // distance
    ]}
    width="max-content"
  >
    <MenuItem>
      Item #1
    </MenuItem>
    <MenuItem>
      Item #2
    </MenuItem>
    <MenuItem>
      Item #3
    </MenuItem>
  </MenuList>
</Menu>
```

### Scrolling

To make the menu scrollable, pass `overflow="auto"` and `maxHeight` to `MenuList`. The menu will become scrollable when the menu items exceed the height of the menu.

```jsx
<Menu>
  <MenuButton variant="secondary">
    <Text>Options</Text>
    <Space width="1x" />
    <MenuIndicator />
  </MenuButton>
  <MenuList
    maxHeight={200}
    overflow="auto"
    width="max-content"
  >
    {Array.from({ length: 100 }).map((_, key) => (
      <MenuItem key={key}>
        Item #{key + 1}
      </MenuItem>
    ))}
  </MenuList>
</Menu>
```

You can also use the `Scrollbar` component to override the default scrollbar.

```jsx
<Menu>
  <MenuButton variant="secondary">
    <Text>Options</Text>
    <Space width="1x" />
    <MenuIndicator />
  </MenuButton>
  <MenuList width="100%">
    <Scrollbar height={200}>
      {Array.from({ length: 100 }).map((_, key) => (
        <MenuItem key={key}>
          Item #{key + 1}
        </MenuItem>
      ))}
    </Scrollbar>
  </MenuList>
</Menu>
```

### Input controls

Set `closeOnSelect` to `false` to keep the menu open after selecting a menu item.

```jsx
function menuWithCheckbox() {
  const [checkedItems, setCheckedItems] = React.useState(['host', 'ip']);
  const handleChange = (e) => {
    const value = e.target.value;
    const items = [...checkedItems];
    items.indexOf(value) === -1 ? items.push(value) : items.splice(items.indexOf(value), 1);
    setCheckedItems(items);
  };
  return (
    <Menu closeOnSelect={false}>
      <MenuButton variant="secondary">
        <Text>Selected ({checkedItems.length})</Text>
        <Space width="1x" />
        <MenuIndicator />
      </MenuButton>
      <MenuList width="max-content">
        <CheckboxGroup size="sm" defaultValue={checkedItems}>
          <MenuItem><Checkbox value="host" onChange={e => handleChange(e)}>Endpoint name</Checkbox></MenuItem>
          <MenuItem><Checkbox value="ip" onChange={e => handleChange(e)}>IP address</Checkbox></MenuItem>
          <MenuItem><Checkbox value="os" onChange={e => handleChange(e)}>Operating system</Checkbox></MenuItem>
        </CheckboxGroup>
      </MenuList>
    </Menu>
  );
}
```

## Accessibility

### Keyboard Interaction

| Key | Action |
| :-- | :----- |
| `Enter` \| `Space` | When `MenuTrigger` or `MenuButton` is focused, opens the menu. |
| `ArrowDown` \| `ArrowUp`| When `MenuList` is focused, moves the focus to the next menu item. |
| `Escape` | When a menu item is focused, closes the menu and moves the focus to the trigger. |

## Props

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| anchorEl | `Element` | | The element to which the menu is attached. |
| autoSelect | `boolean` | `true` | Whether to automatically select the first menu item when the menu is opened. |

| onKeyDown | `function` | | Callback when the user presses a key. |
| onOpen | `function` | | Callback when the menu is opened. |
| onClick | `function` | | Callback when a menu item is clicked. |
| onClose | `function` | | Callback when the menu is closed. |
| placement | `string` | 'bottom-start' | The placement of the menu. One of: 'top', 'top-start', 'top-end', 'bottom', 'bottom-start', 'bottom-end' |
