# useColorMode

`useColorMode` is a custom Hook that gives you access to the current color mode, and a function to change the color mode.

## Import

```js
import { useColorStyle } from '@tonic-ui/react';
```

## Usage

```jsx disabled
import { css, Global } from '@emotion/react';
import { Button, useColorMode } from '@tonic-ui/react';
import React from 'react';

const App = () => {
  const [colorMode, setColorMode] = useColorMode(); // One of: 'dark', 'light'
  const toggleColorMode = () => {
    const nextColorMode = {
      'dark': 'light',
      'light': 'dark',
    }[colorMode];
    setColorMode(nextColorMode);
  };

  return (
    <>
      <Global
        styles={css`
          :root {
            color-scheme: ${colorMode};
          }
        `}
      />
      <Button onClick={toggleColorMode}>
        Toggle Color Mode
      </Button>
    </>
  );
};
```

### Returns

Returns an array with the current color mode and a function to change the color mode.
