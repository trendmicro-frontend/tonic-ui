# Link

A link is an interactive text element that enables users to navigate to another destination — such as a different application, website, resource, or a specific section on the same page.

Links are intended primarily for navigation, not for performing actions.

## Import

```js
import { Link } from '@tonic-ui/react';
```

#### `disabled`

The `disabled` attribute is not natively supported by the anchor (`<a>`) element. The `Link` component adds it for styling purposes only.

## Props

### Link

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| children | ReactNode | | |
| disabled | boolean | | If `true`, the link is displayed as disabled. |
| onClick | function | | A callback called when the link is clicked. |
| variant | string | 'default' | Defines the visual style of the link: <br />- 'default': For standalone links or navigation items.<br />- 'inline': For inline links within text or paragraphs.<br />- 'subtle': For links with reduced visual emphasis but still interactive. |
