# TextLabel

`TextLabel` is a block-level element that composes `Text` and represents as a label.

## Import

```js
import { TextLabel } from '@tonic-ui/react';
```

## Usage

{render('./basic')}

## Props

### TextLabel

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| children | ReactNode | | |
| size | string | | One of: '4xl', '3xl', '2xl', 'xl', 'lg', 'md', 'sm', 'xs' |
