# Mark

The `Mark` component is used to emphasize or draw attention to specific text. It offers multiple variants for different purposes, such as emphasizing content or marking search keywords.

## Import

```js
import { Mark } from '@tonic-ui/react';
```

## Usage

### Variants

The `Mark` component supports the following variants: `highlight`, `emphasis`, and `none`.
If no variant is specified, it defaults to `none`.

## Props

### Mark

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| children | ReactNode | | |
| variant | string | 'none' | The variant to use. One of: 'highlight', 'emphasis', 'none'. |
