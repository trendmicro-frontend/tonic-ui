# Mark

The `Mark` component is used to emphasize or draw attention to specific text. It offers multiple variants for different purposes, such as emphasizing content, marking search keywords, or indicating text selection.

## Import

```js
import { Mark } from '@tonic-ui/react';
```

## Usage

### Variants

The `Mark` component supports the following variants: `none`, `highlight`, `selection`, and `emphasis`.
If no variant is specified, it defaults to `none`.

## Props

### Mark

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| children | ReactNode | | |
| variant | string | 'none' | The variant to use. Defaults to 'none' if not provided. Available options: 'none', 'emphasis', 'highlight', 'selection'. |
