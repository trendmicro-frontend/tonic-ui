# ButtonBox

`ButtonBox` looks and behaves like a button but renders a non-button element (`<div role="button">`). Use it where a native `<button>` is not permitted, such as a button nested inside another button.

## Import

```js
import { ButtonBox } from '@tonic-ui/react';
```

## Keyboard interaction

To support accessibility and expected user experience, `ButtonBox` emulates the keyboard behavior of a native button:

- **Enter key**:
  - Triggers the `onClick` handler on **keydown**.
  - Ignores auto-repeat events (`event.repeat`) to avoid firing multiple times while the key is held down.

- **Space key**:
  - Prevents the default scrolling behavior on **keydown**.
  - Triggers the `onClick` handler on **keyup**, matching the native `<button>` behavior.
  - Ignores auto-repeat events (`event.repeat`) to avoid unintentional repeated clicks.

## Props

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| as | string \| ComponentType | 'div' | The HTML element or component to render. |
| children | node | | The content of the button box. |
| disabled | boolean | false | If `true`, the control is disabled. Sets `aria-disabled`; styleable via `_disabled`. |
| onClick | function | | Callback fired when activated by a mouse click, the Enter key, or the Space key. |
| tabIndex | number | 0 | The tab order. Removed (not focusable) when `disabled`. |
