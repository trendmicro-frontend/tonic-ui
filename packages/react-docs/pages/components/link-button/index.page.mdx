# LinkButton

This component provides an element resembling an anchor that purely acts as a click target with no navigation as result. It can also be used inline in text.

## Import

```js
import { LinkButton } from '@tonic-ui/react';
```

## Usage

{render('./basic')}

## Props

### LinkButton

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| children | ReactNode | | |
| disabled | boolean | | If `true`, the link is displayed as disabled. |
| onClick | function | | A callback called when the link is clicked. |
| variant | string | 'default' | Defines the visual style of the link: <br />- 'default': For standalone links or navigation items.<br />- 'inline': For inline links within text or paragraphs.<br />- 'subtle': For links with reduced visual emphasis but still interactive. |
