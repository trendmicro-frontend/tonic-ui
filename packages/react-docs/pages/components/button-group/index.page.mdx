# ButtonGroup

`ButtonGroup` is used to group related buttons.

## Import

```js
import { ButtonGroup } from '@tonic-ui/react';
```

## Usage

### Basic button group

{render('./basic')}

### Group variants

Use the `variant` prop to change the visual style of every button in a group. You can set the value to `emphasis`, `primary`, `default`, `secondary` and `ghost`.

{render('./variants')}

### Group sizes

Use the `size` prop to change the size of the `ButtonGroup`. You can set the value to `sm`, `md`, or `lg`.

{render('./sizes')}

### Group orientation

Make a set of buttons appear vertically stacked rather than horizontally, by adding `orientation="vertical"` to the `ButtonGroup` component.

{render('./orientation')}

### Group states

The following example shows different states (i.e. normal, disabled, and selected states) when composing a group of buttons.
* Use the `disabled` prop to make the `Button` appear disabled.
* Use the `selected` prop to make the `Button` appear selected.

{render('./states')}

## Props

### ButtonGroup

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| children | ReactNode \| `(context) => ReactNode` | | A function child can be used intead of a React element. This function is called with the context object. |
| orientation | string | 'horizontal' | The orientation of the button group. One of: 'horizontal', 'vertical' |
| size | string | 'md' | The size of the button group. One of: 'sm', 'md', 'lg' |
| variant | string | 'default' | The variant of the button group. One of: 'emphasis', 'primary', 'default', 'secondary', 'ghost' |
