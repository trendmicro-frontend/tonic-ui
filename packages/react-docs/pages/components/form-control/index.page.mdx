# FormControl

A comprehensive form control system that provides accessible, composable form fields with automatic context management and consistent styling.

## Import

- `FormControl`: Provides context and wrapper structure, managing form field state and accessibility attributes.
- `FormLabel`: Label component with required indicator support and automatic association with form fields.
- `FormInput`: Input component that automatically associates with label, error message, helper text, and character count.
- `FormTextarea`: Textarea component with automatic form control context integration.
- `FormHelperText`: Helper text component that provides form guidance.
- `FormErrorMessage`: Error message display supporting single or multiple errors.
- `FormCharacterCount`: Character counter displaying character limits and over-limit states.
- `useFormControl`: Hook to access FormControl context.

```js
import {
  FormControl,
  FormLabel,
  FormInput,
  FormTextarea,
  FormHelperText,
  FormErrorMessage,
  FormCharacterCount,
  useFormControl,
} from '@tonic-ui/react';
```

## Accessibility

- `FormControl` automatically handles ID generation, ensuring label and input are properly associated using `htmlFor` and `id` attributes.
- Error messages automatically set `role="alert"` for screen readers.

### ARIA Attributes

FormControl automatically manages the following ARIA attributes:
- `aria-invalid`: Automatically set when `error={true}`.
- `aria-describedby`: Automatically includes `formHelperTextId`, `formErrorMessageId`, and `formCharacterCountId`.
- `aria-disabled`: Automatically set when `disabled={true}`.
- `aria-readonly`: Automatically set when `readOnly={true}`.

## Props

### FormControl

| Prop        | Type                       | Default    | Description                                                        |
| ----------- | -------------------------- | ---------- | ------------------------------------------------------------------ |
| error       | boolean                    | false      | If `true`, passes error state to child form elements that support the error prop (e.g., `FormInput`, `FormTextarea`), and displays `FormErrorMessage`. |
| disabled    | boolean                    | false      | If `true`, passes disabled state to child form elements that support the disabled prop (e.g., `FormInput`, `FormTextarea`). |
| readOnly    | boolean                    | false      | If `true`, passes read-only state to child form elements that support the readOnly prop (e.g., `FormInput`, `FormTextarea`). |
| orientation | 'horizontal' \| 'vertical' | 'vertical' | The layout orientation of the form                                 |
| children    | ReactNode                  | -          | Child component content                                            |

### FormLabel

| Prop     | Type      | Default | Description                                              |
| -------- | --------- | ------- | -------------------------------------------------------- |
| required | boolean   | false   | Whether to display the required indicator (red asterisk) |
| children | ReactNode | -       | Label text content                                       |

### FormHelperText

| Prop     | Type      | Default | Description         |
| -------- | --------- | ------- | ------------------- |
| children | ReactNode | -       | Helper text content |

### FormErrorMessage

| Prop   | Type                     | Default | Description                                     |
| ------ | ------------------------ | ------- | ----------------------------------------------- |
| errors | ReactNode[] \| ReactNode | []      | Error messages, supports string or string array |

### FormCharacterCount

| Prop     | Type   | Default | Description                      |
| -------- | ------ | ------- | -------------------------------- |
| count    | number | 0       | Current input character count.   |
| maxCount | number | 0       | Maximum allowed character count. |
