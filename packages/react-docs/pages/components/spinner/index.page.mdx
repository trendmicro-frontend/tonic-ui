# Spinner

A spinner can be used to display the loading state when the part of the page is waiting for action results.

## Import

```js
import { Spinner } from '@tonic-ui/react';
```

## Usage

{render('./basic')}

### Sizes

You can use the `size` prop to change the size of the spinner. Each size has a cooresponding dimension and stroke width as shown below. If the `size` prop is not provided, it will default to `md`.

| Size | Dimension (px) | Thickness (px) |
| :--- | :------------- | :------------- |
| xs | 16 x 16 | 2 |
| sm | 32 x 32 | 2 |
| md | 48 x 48 | 4 |
| lg | 64 x 64 | 4 |
| xl | 80 x 80 | 4 |

{render('./sizes')}

### Colors

* `color` - Defines the spinner's indicator color. Use `transparent` to hide the indicator.
* `trackColor` - Defines the spinner's track color. Use `transparent` to hide the track.

{render('./colors')}

## Props

### Spinner

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| color | string | 'blue:60' | The color of the spinner. It supports both default and custom theme colors. |
| size | string | 'md' | The size of the spinner. One of: 'xl', 'lg', 'md', 'sm', 'xs'. |
| thickness | number | 4 | The thickness of the spinner, in pixels. |
| trackColor | string | Dark mode:<br/>'rgba(255, 255, 255, .12)'<br/><br/>Light mode:<br/>'rgba(0, 0, 0, .12)' | The spinner's track color. |
