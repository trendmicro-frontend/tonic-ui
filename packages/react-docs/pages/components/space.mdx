# Space

Space is [Box](box) with `display="inline-block"`. You can specify either `width` or `height` to add space between two elements.

## Import

```js
import { Space } from '@tonic-ui/react';
```

## Usage

### Horizontal spacing

```jsx
<>
  <Button variant="ghost" width="8x" height="8x" borderRadius="circle">
    <Icon icon="add" />
  </Button>
  <Space width="2x" />
  <Button variant="ghost" borderRadius="2rem">
    <Icon icon="settings" />
    <Space width="2x" />
    Settings
  </Button>
</>
```

### Vertical spacing

```jsx
<Flex flexDirection="column">
  <TextLabel>Text label</TextLabel>
  <Space height="2x" />
  <Input placeholder="Text input" />
  <Space height="1x" />
  <Text size="xs">Help text for the text input</Text>
</Flex>
```
