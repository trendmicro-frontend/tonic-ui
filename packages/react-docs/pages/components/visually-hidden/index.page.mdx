# VisuallyHidden

`VisuallyHidden` is a technique that addresses accessibility concerns. It visually hides the content of the element, but keep it readable for screen readers.

## Import

```js
import { VisuallyHidden } from '@tonic-ui/react';
```

## Usage

The text `Checkmark` is visually hidden from the icon button, but it is still readable by screen readers.
