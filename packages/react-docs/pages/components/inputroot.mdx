# InputRoot

`InputRoot` is basically the same as `Input` but it allows you to add adornments to the input.

## Import

```js
import {
  InputRoot,
  InputAdornment,
} from '@tonic-ui/react';
```

## Usage

```jsx
<>
  <TextLabel mb="1x">Label:</TextLabel>
  <InputRoot placeholder="Basic example" />
  <Text size="xs" mt="1x">Help text for the text input</Text>
</>
```

> `InputRoot` is built on top of `Input`. Visit the [Input](input) page for more information.

### Input adornments

```jsx disabled
<InputRoot
  placeholder="yyyy-MM-dd"
  defaultValue={new Date().toISOString().substr(0, 10)}
  startAdornment={(
    <InputAdornment>
      <Icon icon="calendar" />
    </InputAdornment>
  )}
  endAdornment={(
    <InputAdornment>
      <Icon icon="check" />
    </InputAdornment>
  )}
/>
```

The main way to prepend or append an input adornment is to use the `InputAdornment` component. This can be used to add a prefix, a suffix, or an action to an input. For instance, you can use an icon button to hide or reveal the password.

```jsx
function Example() {
  const [view, setView] = React.useState(false);
  const toggleView = () => setView(view => !view);
  const [colorMode] = useColorMode();
  const adornmentColor = {
    dark: 'white:tertiary',
    light: 'black:tertiary',
  }[colorMode];
  const adornmentIconName = view ? 'view' : 'view-off';
  const inputType = view ? 'text' : 'password';

  return (
    <Flex direction="column" rowGap="4x">
      <InputRoot
        placeholder="Mobile phone number"
        startAdornment={(
          <InputAdornment>
            <Icon icon="mobile" color={adornmentColor} />
          </InputAdornment>
        )}
      />
      <InputRoot
        placeholder="Enter the amount"
        startAdornment={(
          <InputAdornment>
            <Text width="4x" textAlign="center" color={adornmentColor}>$</Text>
          </InputAdornment>
        )}
        endAdornment={(
          <InputAdornment>
            <Icon icon="check" color="green:50" />
          </InputAdornment>
        )}
      />
      <InputRoot
        type={inputType}
        defaultValue="Revealed Password"
        placeholder="Password"
        endAdornment={(
          <InputAdornment>
            <ButtonBase onClick={toggleView}>
              <Icon icon={adornmentIconName} color={adornmentColor} />
            </ButtonBase>
          </InputAdornment>
        )}
      />
    </Flex>
  );
}
```

## Props

### InputRoot

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| disabled | boolean | | If `true`, the user cannot interact with the control. This sets `aria-disabled=true` and you can style this state by passing the `_disabled` prop. |
| endAdornment | ReactNode | | End `InputAdornment` for this component. |
| isInvalid | boolean | | If `true`, the input will indicate an error. You can style this state by passing the `_invalid` prop. |
| readOnly | boolean | | If `true`, prevents the value of the input from being edited. |
| size | string | 'md' | The visual size of the `input` element. One of: 'sm', 'md', 'lg' |
| variant | string | 'outline' | The variant of the input style to use. One of: 'outline', 'filled', 'unstyled' |
| startAdornment | ReactNode | | Start `InputAdornment` for this component. |
