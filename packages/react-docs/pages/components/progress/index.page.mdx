# Progress

Progress indicators inform users about the status of ongoing processes, such as loading an app, submitting a form, or saving updates.

- **Determinate** indicators display how long an operation will take.
- **Indeterminate** indicators visualize an unspecified wait time.

## Import

```js
import {
  CircularProgress,
  LinearProgress,
} from '@tonic-ui/react';
```

## Props

### CircularProgress

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| color | string | 'blue.600' | The color of the circular progress. It supports both default and custom theme colors. |
| min | number | 0 | The minimum value of the progress indicator. |
| max | number | 100 | The maximum value of the progress indicator. |
| size | number | 48 | The diameter of the circular progress, in pixels. |
| thickness | number | 4 | The thickness of the circular progress, in pixels. |
| value | number | 0 | The value of the progress indicator for the determinate variant. Value between `min` and `max`. |
| variant | string | 'indeterminate' | The variant to use. One of: 'indeterminate', 'determinate'<br />• Use 'indeterminate' when there is no progress value.<br />• Use 'determinate' when a progress value is defined. |

> ℹ️  To scale the circular progress, set the `width` and `height` props to identical values. For example, if the `size` is 48, passing 96 to both `width` and `height` will scale the circular progress by 2x.
