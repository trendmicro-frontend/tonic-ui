# SearchDropdown

A search dropdown component based on `Dropdown` and `SearchInput`, designed to render a menu with selectable items and handle selection logic through search functionality.

## Usage

### Basic configuration

Here's the simplest configuration to get started with the `SearchDropdown` component.

```jsx
<SearchDropdown
  onSelect={(item) => {
    setItem(item);
  }}
  items={[
    { label: 'Item 1', value: 1 },
    { label: 'Item 2', value: 2 },
  ]}
  renderContent={({ items, renderItems, renderSearchInput }) => (
    <>
      <Box px="3x" mb="2x">
        {renderSearchInput()}
      </Box>
      <Scrollbar
        maxHeight={36 * 5}
        overflowY="visible"
      >
        {renderItems(items)}
      </Scrollbar>
    </>
  )}
  renderItem={(item, { searchKeyword }) => {
    return item.label;
  }}
>
  {item.label}
</SearchDropdown>
```
