# Dropdown

A dropdown component built on `DropdownBase` and `MenuButton` that renders a menu with selectable options and manages the selection behavior.

## Usage

Here's the simplest configuration:

```jsx
<Dropdown
  onSelect={(option) => {
    setOption(option);
  }}
  options={[
    { value: 1, label: 'Option 1' },
    { value: 2, label: 'Option 2' },
  ]}
>
  {option.label}
</Dropdown>
```
