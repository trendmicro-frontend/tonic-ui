# ButtonBox

`ButtonBox` is a custom component designed to function and appear like a button. It can be used in contexts where native `<button>` elements are not permitted, such as within other buttons.

## Keyboard interaction

To support accessibility and expected user experience, `ButtonBox` emulates the keyboard behavior of a native button:

- **Enter key**:
  - Triggers the `onClick` handler on **keydown**.
  - Ignores auto-repeat events (`event.repeat`) to avoid firing multiple times while the key is held down.

- **Space key**:
  - Prevents the default scrolling behavior on **keydown**.
  - Triggers the `onClick` handler on **keyup**, matching the native `<button>` behavior.
  - Ignores auto-repeat events (`event.repeat`) to avoid unintentional repeated clicks.

You can utilize the following utility function to create a `ButtonBox` that supports variants:

```js
const EmphasisButtonBox = createVariantButtonBox('emphasis');
const PrimaryButtonBox = createVariantButtonBox('primary');
const DefaultButtonBox = createVariantButtonBox('default');
const SecondaryButtonBox = createVariantButtonBox('secondary');
const GhostButtonBox = createVariantButtonBox('ghost');
```
