# Dropdown

A dropdown component built on `DropdownBase` and `MenuButton` that renders a menu with selectable items and manages the selection behavior.

## Examples

### Basic configuration

Here's the simplest configuration to get started with the `Dropdown` component.

```jsx
<Dropdown
  onSelect={(item) => {
    setItem(item);
  }}
  items={[
    { label: 'Item 1', value: 1 },
    { label: 'Item 2', value: 2 },
  ]}
>
  {item.label}
</Dropdown>
```

* `items`: An array of selectable items. Each item typically includes a `label`, along with an optional identifier such as `id` or `value` of any type. The structure is flexible and can be customized to fit your specific needs.
* `onSelect`: A callback function that receives the selected item and can be used to update the component state.
* `children`: The content rendered within the dropdown trigger. In this example, it shows the `label` of the currently selected item.

### Custom dropdown toggle

By default, the dropdown uses a `MenuButton` as its toggle. To customize the toggle, provide your own component (wrapped with `forwardRef`) using the `slots.toggle` prop:

```jsx
<Dropdown
  onSelect={onSelect}
  items={items}
  slots={{
    toggle: CustomDropdownToggle,
  }}
  slotProps={{
    toggle: {
      // Additional props to pass to the toggle component
    },
  }}
>
  {selectedItem.label}
</Dropdown>
```
