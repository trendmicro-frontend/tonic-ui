# Dropdown

A dropdown component built on `DropdownBase` and `MenuButton` that renders a menu with selectable items and manages the selection behavior.

## Examples

### Basic

* `defaultValue` / `value`: Use `defaultValue` for uncontrolled mode, or `value` paired with `onChange` for controlled mode.
* `items`: An array of selectable items. Each item typically includes a `label`, along with an optional identifier such as `value` of any type.
* `onChange`: Called with the selected item whenever the selection changes.

```jsx
// Only the `label` property is required. Other fields are optional.
const items = [
  { id: 'all', label: 'All' },
  { id: 1, label: 'Item 1' },
  { id: 2, label: 'Item 2' },
];

const [value, setValue] = useState(items[0]);

<Dropdown
  defaultValue={items[0]} // [optional] Initial value
  items={items}
  onChange={setValue}
/>
```

You can use `renderToggle` to customize the dropdown toggle. By default, `Dropdown` renders a `MenuButtonToggle` that displays the selected item label.

Additional props can be passed to `getToggleProps` to control the toggle behavior and apply attributes:

```jsx
renderToggle={({ getToggleProps, renderItem, value }) => {
  const toggleProps = {
    'data-test': 'dropdown-toggle',
    disabled: false,
  };
  return (
    <MenuButtonToggle {...getToggleProps(toggleProps)}>
      {renderItem(value)}
    </MenuButtonToggle>
  );
}}
```

### Custom dropdown toggle

By default, the dropdown uses a `MenuButton` as its toggle. To customize the toggle, provide a `renderToggle` callback. The callback receives `getToggleProps`, `renderItem`, and `value` so you can wire interaction props and display the selection:

```jsx
<Dropdown
  items={items}
  defaultValue={items[0]}
  onChange={setValue}
  renderToggle={({ getToggleProps, renderItem, value }) => {
    const toggleProps = {
      'data-test': 'dropdown-toggle',
      disabled: false,
      isClosable: true,
    };
    return (
      <TagToggle {...getToggleProps(toggleProps)}>
        {renderItem(value)}
      </TagToggle>
    );
  }}
/>
```

Pass user-supplied props (e.g. `disabled`, `isClosable`) through `getToggleProps(userProps)` — it merges them with the internal handlers and ensures click suppression when `disabled` is set.
