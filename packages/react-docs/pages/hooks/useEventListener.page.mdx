# useEventListener

A custom Hook to manage browser event listeners.


## Import

```js
import { useEventListener } from '@tonic-ui/react-hooks';
```

## Usage

```js
useEventListener(target, eventName, eventHandler, [options]);
```

### Parameters

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| target | HTMLElement | | The target element to attach the event listener to. |
| eventName | string | | The name of the event to listen for. |
| eventHandler | `(event: MouseEvent) => void` | | The event handler to call when the event is fired. |
| options? | object | | Optional options to pass to the event handler. |
