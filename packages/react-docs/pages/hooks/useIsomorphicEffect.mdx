# useIsomorphicEffect

A custom Hook that resolves to `useEffect` when "window" is not in scope and `useLayoutEffect` in the browser.

```js
useIsomorphicEffect(effect, dependencies);
```

### Parameters

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| effect | function | | The effect to run. |
| dependencies | array | | The dependencies to watch. |

### Example

```jsx
function Component() {
  useIsomorphicEffect(() => {
    console.log('useIsomorphicEffect executed');
  }, []);

  return (
    <Text>
      <Code>useIsomorphicEffect()</Code> resolves to <Code>{useIsomorphicEffect === React.useEffect ? 'useEffect()' : 'useLayoutEffect()'}</Code>
    </Text>
  );
}
```
