# useOnceWhen

A custom Hook that runs a callback at most once when a condition becomes true.

```js
// import
import { useOnceWhen } from '@tonic-ui/react-hooks';

// usage
useOnceWhen(callback, when);
```

### Parameters

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| callback | `() => void` | | The callback to run. |
| when | boolean | false | The condition to run the callback. |

### Example

```jsx
function Component() {
  const [value, setValue] = React.useState(0);
  const ref = React.useRef(0);

  useOnceWhen(() => {
    console.log('This will run only once when clicked');
    ref.current++;
  }, (value > 0));

  return (
    <>
      <Box mb="4x">
        Callback called: {ref.current}
      </Box>
      <Button onClick={() => setValue(value => value + 1)}>
        Click Me
      </Button>
    </>
  );
}
```
