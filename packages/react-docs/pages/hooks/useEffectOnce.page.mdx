# useEffectOnce

A modified version of the `useEffect` Hook that runs an effect only once.

## Import

```js
import { useEffectOnce } from '@tonic-ui/react-hooks';
```

## Usage

```js
useEffectOnce(effect);
```

### Parameters

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| effect | `() => void` | | The effect to run. |
