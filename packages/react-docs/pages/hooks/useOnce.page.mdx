# useOnce

A custom Hook that runs a callback at most once.

## Import

```js
import { useOnce } from '@tonic-ui/react-hooks';
```

## Usage

```js
useOnce(callback);
```

### Parameters

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| callback | `() => void` | | The callback to run. |
