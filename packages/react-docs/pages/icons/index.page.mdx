# React Icons

Tonic icons distributed as SVG React components.

## Installation

```bash
// with npm
npm install @tonic-ui/react-icons@2.x

// with yarn
yarn add @tonic-ui/react-icons@2.x
```

## Usage

```jsx
import { AddIcon } from '@tonic-ui/react-icons';

// Render the icon component
<AddIcon />
<AddIcon color="red:50" />
<AddIcon color={colorStyle.color.secondary} />
```
