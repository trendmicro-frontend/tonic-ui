# CSSBaseline

The CSSBaseline component normalizes styles using [normalize.css](https://github.com/necolas/normalize.css) that provides better cross-browser consistency in the defauly styling of HTML elements.

## Import

```js
import CSSBaseline from '@trendmicro/react-styled-core/CSSBaseline';
// or
import { CSSBaseline } from '@trendmicro/react-styled-core';
```

## Usage

```jsx readonly
<React.Fragment>
  <CSSBaseline />
  {/* The rest of your application */}
</React.Fragment>
```
