# Borders

* `borders`
* `radii` (`border-radius`)

## Border

Tonic UI does not define the border property by default.

<ThemeParser theme="borders" />

The width of the border can be either an explicit nonnegative `<length>` or a keyword. If it's a keyword, it must be one of the following values:
* `thin`
* `medium`
* `thick`

Note: Because the specification doesn't define the exact thickness denoted by each keyword, the precise result when using one of them is implementation-specific. Nevertheless, they always follow the pattern `thin ≤ medium ≤ thick`, and the values are constant within a single document.

### Nonnegative length

```jsx noInline
const Block = (props) => {
  const [colorMode] = useColorMode();
  const backgroundColor = {
    dark: 'gray:90',
    light: 'gray:10',
  }[colorMode];
  return (
    <Flex
      width="32x"
      height="12x"
      alignItems="center"
      justifyContent="center"
      backgroundColor={backgroundColor}
      {...props}
    />
  );
};

function Example() {
  return (
    <Stack direction="row" spacing="6x">
      <Block
        borderWidth={1}
        borderStyle="solid"
        borderColor="gray:50"
      >
        border
      </Block>
      <Block
        borderTopWidth="1x"
        borderTopStyle="solid"
        borderTopColor="gray:50"
      >
        borderTop
      </Block>
      <Block
        borderRightWidth="1x"
        borderRightStyle="solid"
        borderRightColor="gray:50"
      >
        borderRight
      </Block>
      <Block
        borderBottomWidth="1x"
        borderBottomStyle="solid"
        borderBottomColor="gray:50"
      >
        borderBottom
      </Block>
      <Block
        borderLeftWidth="1x"
        borderLeftStyle="solid"
        borderLeftColor="gray:50"
      >
        borderLeft
      </Block>
    </Stack>
  );
}

render(<Example />);
```

## Border Radius

<ThemeParser theme="radii" />

### `borderRadius="circle"`

```jsx
<Box
  backgroundColor="gray:80"
  border={1}
  borderColor="gray:40"
  borderRadius="circle"
  width="18x"
  height="18x"
/>
```
