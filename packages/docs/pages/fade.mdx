# Fade

Fade in from transparent to opaque.

This component uses [react-spring](https://github.com/react-spring/react-spring) to create animation.

## Import

```js
import { Fade } from '@tonic-ui/react';
```

## Usage

### Basic usage

```jsx
function Example() {
  const [show, setShow] = React.useState(false);
  const handleToggle = () => setShow(!show);

  return (
    <Box whiteSpace="pre-wrap">
      <Box mb="2x">
        <Button variantColor="green" onClick={handleToggle}>
          Toggle
        </Button>
      </Box>
      <Box height={64}>
        <Fade show={show}>
```

## Props

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| show | boolean | false | If `true`, the content will be visible. |
| duration | number | 200 | The duration for the transition, in milliseconds. |
